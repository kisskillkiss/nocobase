```ts
}
```

## app.i18n

国际化，i18next 实例，详情见 [I18next API](https://www.i18next.com/overview/api)，在 Middleware 里请使用 [ctx.i18n](context#ctxi18n)

##### Definition

```ts
class Application {
  public i18n: I18next.i18n;
}
```

##### Examples

```ts
app.i18n.t('Hello');
```
